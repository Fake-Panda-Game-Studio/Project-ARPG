1. Open folder data
2. Copy all files from data folder to $YOUR PROJECT FOLDER$\data\
 Files to copy:
	-  PKD_MapInventorySettings.json
	-  PKD_UserChestSettings.json
	-  PKD_MapChestSettings.json

3. Open folder img
4. Copy folder pMapInventory to $YOUR PROJECT FOLDER$\img\

If you using RPG Maker MV
	5. Copy PKD_MapInventory.js (from folder MV) to $YOUR PROJECT FOLDER$\js\plugins\
If you using PRG Maker MZ
	5. Copy PKD_MapInventory.js (from folder MZ) to $YOUR PROJECT FOLDER$\js\plugins\

6. Plug PKD_MapInventory in Plugin Manager
